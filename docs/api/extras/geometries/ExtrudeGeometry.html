[name]

Creates extruded geometry from a path shape

Constructor

[name]([page:Array shapes], [page:Object options])

shapes — A Shape or an array of shapes (THREE.SHAPE Objects).
options — Object that can contain the following parameters.
  • curveSegments — int. The number of segments that will be used when extruding a shape's curve. The more segments used, the smoother the curves will look
  • steps — int. The number of segments along the depth of the extrusion into which the extrusion will be divided. A higher value will result in more individual faces. The default value is 1.
  • amount — int. How far (the depth) the shape should be extruded
  • bevelEnabled — bool. turn on bevel
  • bevelThickness — float. The depth of the bevel (i.e.: how deep into the original shape bevel goes). The bevel is the rounded corner between the front and back faces and the extrusion
  • bevelSize — float. Distance between the bevel and the shape outline. This determines the height of the bevel.
  • bevelSegments — int. The number of segments (layers) that will be use by the bevel. The greater the number of segments used, the smoother the bevel will look.
  • extrudePath — THREE.CurvePath. 3d spline path to extrude shape along. (creates Frames if (frames aren't defined) If this isn't specifed, the shape is extruded along the z-axis.
  • frames — THREE.TubeGeometry.FrenetFrames. containing arrays of tangents, normals, binormals
  • material — int. material index for front and back faces
  • extrudeMaterial — int. material index for extrusion and beveled faces
  • uvGenerator — Object. object that provides UV generator functions
This object extrudes an 2D shape to an 3D geometry.
